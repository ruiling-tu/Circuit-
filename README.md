# Circuit

Circuit is a minimalist iOS reset app that helps users break unhelpful mental loops in 60–90 seconds through a structured flow: state selection, distortion recognition, reframe, physiological reset, and a micro-action anchor.

## Features
- SwiftUI + MVVM architecture
- SwiftData persistence (local only)
- Gentle, time-based notification nudges
- Haptics and breathing animation
- History with streak and weekly stats

## Requirements
- iOS 17+
- Xcode 15+

## Build & Run
1. Open `Circuit.xcodeproj` in Xcode.
2. Select a simulator or physical device.
3. Run the app.

## Privacy
All data stays on device. No accounts, no cloud, no analytics sent.
